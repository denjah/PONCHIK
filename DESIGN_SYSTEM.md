# Design System: Neobrutalism (Ponchic)

## 1. Core Philosophy
Этот проект использует стиль **Необрутализм (Neobrutalism)**, вдохновленный поп-артом 90-х и стикер-артом. Дизайн должен быть энергичным, дерзким, молодежным, с контролируемым хаосом поверх строгой сетки.

## 2. Design Tokens (CSS Variables / Tailwind)

### 2.1 Colors (Палитра)
Открытые, кричащие, «кислотные» цвета, сталкивающиеся на максимальном контрасте. Никаких градиентов или пастельных тонов (кроме фона).
- **Background**: `#FDFBF7` (Off-white / Paper) — теплый фон, на котором контрастируют яркие элементы.
- **Primary / Brand**: `#FF00FF` (Acid Pink / Пончиковый розовый)
- **Secondary**: `#00FF00` (Lime Green / Ядовито-салатовый)
- **Accent 1**: `#FFFF00` (Electric Yellow)
- **Accent 2**: `#00FFFF` (Cyan)
- **Dark/Stroke**: `#1A1A1A` (Почти черный для обводок и теней)

### 2.2 Strokes (Глобальная толстая обводка)
Почти каждый элемент (кнопки, карточки, текст, картинки) должен иметь жесткую черную обводку, создавая эффект стикера или комикса.
- **Border Width (Standard)**: `3px`
- **Border Width (Thick - for main cards/hero)**: `4px`
- **Border Color**: `#1A1A1A`

### 2.3 Hard Shadows (Жесткие тени)
Никакого размытия (blur: 0). Тени — это сплошные плашки, сдвинутые по осям X и Y. Это дает грубый, плоский псевдо-3D эффект.
- **Shadow Sm**: `2px 2px 0px 0px #1A1A1A`
- **Shadow Md**: `4px 4px 0px 0px #1A1A1A`
- **Shadow Lg**: `8px 8px 0px 0px #1A1A1A`
- **Shadow Hover (Interactive)**: Смещение элемента и уменьшение тени, чтобы создать физическое ощущение нажатия (напр. смещение элемента на 2px вниз/вправо и уменьшение тени до `2px 2px 0px 0px #1A1A1A`).

### 2.4 Typography (Крупная акцидентная типографика)
Шрифты крупные, с имитацией ручной надписи, маркера или старых плакатов.
- **Heading Font**: `'Archivo Black', system-ui, sans-serif` (жесткий, толстый, без засечек).
- **Body Font**: `'Space Grotesk', system-ui, sans-serif` (для читаемости, но с характером).
- **Text Styling**: Заголовки часто с `text-transform: uppercase`, иногда с небольшим поворотом (`rotate(-2deg)` или `rotate(2deg)`) для небрежности.
- Текст поверх картинок часто имеет жесткий text-shadow: `-2px -2px 0 #1A1A1A, 2px -2px 0 #1A1A1A, -2px 2px 0 #1A1A1A, 2px 2px 0 #1A1A1A`.

### 2.5 Geometry & Spacing (Асимметрия и сетка)
- Контент удерживается в строгой сетке (Grid/Flexbox).
- Внешняя небрежность достигается за счет небольших вращений (`rotate(-1deg)`), смещений карточек и плашек, наезжающих друг на друга слоев (z-index).
- **Border Radius**: Легкое скругление для эффекта стикера, либо вообще без него (острые углы). Мы используем скругление стикера: `border-radius: 8px` или `12px` для кнопок и карточек, но жестко.

## 3. UI Components Rules
- **Buttons**: Толстая обводка (3px), яркий фон (например, Acid Pink), жесткая тень (4px 4px). При hover тень может становиться цветной или элемент может сдвигаться (micro-interactions из `design-spells`).
- **Cards**: Белый или желтый фон, толстая черная обводка, тени без блюра. Внутри — крупный текст и стилизованные под клипарт изображения.
- **Backgrounds**: Паттерны в стиле поп-арт (горошек / halftone pattern, лучи, шахматка) с низкой непрозрачностью.
